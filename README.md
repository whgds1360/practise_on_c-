# Практика по C# (practise_on_c#)

**Учебный репозиторий для изучения C# и разработки десктопных приложений**

Привет!👋  
Это мой личный репозиторий, где я практикуюсь в программировании на **C#**.  
Я учусь создавать Windows-приложения и уже реализовал **три полноценных проекта** на WinForms.

---

## 🎯 Что здесь есть

### Реализованные проекты:

1. **Чистый WinForms**  
   Классическое приложение только на стандартных компонентах .NET Windows Forms.  
   Без сторонних библиотек — чистый код, работа с формами, событиями, элементами управления и логикой.

2. **WinForms + Guna 2 Framework**  
   Современное приложение с красивым и современным интерфейсом.  
   Использован фреймворк **Guna UI 2** (анимации, кастомные кнопки, панели, графики, темы и т.д.).  
   Показывает, как быстро делать красивые и профессиональные десктопные приложения.

   Калькулятор с современным интерфейсом

## 🛠 Технологии и инструменты

- **Язык:** C#  
- **Фреймворк:** .NET Framework / .NET (Windows Forms)  
- **UI:**  
  - Стандартный WinForms  
  - Guna.UI2.WinForms (для второго проекта)
- **IDE:** Visual Studio
- **Дополнительно:** работа с событиями, компонентами, ресурсами
